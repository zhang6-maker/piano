一个结合了星空动画与钢琴演奏的交互式网页应用，提供沉浸式的音乐视觉体验。

✨ 功能特色
🎨 视觉特效
动态星空背景：包含200+闪烁的星星，亮度随机变化

蓝色烟雾粒子：按键时产生蓝色烟雾上升动画，效果不够炫酷

🎹 钢琴功能
49键专业钢琴：覆盖C2到C6四个完整八度加一个C

双操作模式：支持鼠标点击和键盘按键演奏

物理模拟按键：按键时有真实的按压动画

智能音符管理：自动处理音符重叠和释放

音效处理：三角波音色 + 混响效果

⌨️ 键盘映射
四区域布局：高中低音区分别映射不同键盘区域

直观指引：侧边栏显示完整的键盘映射指南

丰富的快捷键：支持数字键、字母键、功能键等多种输入方式

🚀 快速开始
在线体验
直接将HTML代码保存为.html文件，在浏览器中打开即可使用。

依赖项
Tone.js (v14.8.49) - CDN引入

现代浏览器（支持Canvas和Web Audio API）

🎮 使用指南
演奏方式
鼠标演奏：直接点击屏幕底部的钢琴键

键盘演奏：使用键盘按键对应不同音符

键盘映射参考
音区	键盘位置	对应按键
高音区 (C5-C6)	数字行	1!2@3 4$5%6^7 8
中音区 (C4-B4)	QWERTY行	QWERT YUI OP[]
中低音区 (C3-B3)	ASDF行	ASDFG HJK L;'↵
低音区 (C2-B2)	ZXCV行	ZXCVB NM,./ `⇥ |
音频初始化
首次使用需要点击页面任意位置激活音频上下文（现代浏览器安全策略要求）。

🛠️ 技术架构

关键技术
Canvas动画：使用requestAnimationFrame实现流畅星空动画

Web Audio API：通过Tone.js进行音频合成和处理

CSS动画：按键效果和烟雾粒子使用CSS Transition/Animation

响应式设计：自适应不同屏幕尺寸

音频配置
合成器：三角波振荡器

包络：Attack: 0.02s, Decay: 0.1s, Sustain: 0.3s, Release: 0.3s

效果：1秒混响时间

音符时长：自动播放1秒后停止



🔧 自定义选项
修改钢琴键数
在setupPiano()方法中调整whiteKeys和blackKeys数组可以改变钢琴的键数范围。

调整视觉效果
星星数量：修改createStars()中的numStars

流星概率：调整createShootingStar()中的随机数阈值

烟雾粒子：修改createSmokeParticles()中的粒子数量

更改音色
在setupAudio()方法中修改Tone.Synth的配置参数：

oscillator.type: 可改为"sine"、"square"、"sawtooth"

envelope: 调整各阶段时长
